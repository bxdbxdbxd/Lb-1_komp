## Описание проекта
Разработанный текстовый редактор представляет собой Windows-приложение с интуитивно понятным интерфейсом, позволяющее выполнять базовые операции с текстовыми файлами. Редактор поддерживает создание, открытие и сохранение файлов, стандартные операции редактирования текста, а также содержит справочную систему.

### Основные возможности:
- Создание нового текстового документа
- Открытие существующего текстового файла
- Сохранение документа (в текущий файл или как новый)
- Редактирование текста (вырезать, копировать, вставить, удалить)
- Отмена и повтор действий
- Выделение всего текста
- Контекстная справка по всем функциям
- Информация о программе

### Команды для сборки проекта

#### Сборка через Visual Studio
1. Откройте решение `WinFormsApp1.sln` в Visual Studio
2. Выберите конфигурацию "Release"
3. Нажмите `Ctrl+Shift+B` или выберите "Build" → "Build Solution"
